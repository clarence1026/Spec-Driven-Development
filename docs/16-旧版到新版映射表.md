# 旧版到新版映射表

## 作用

本表用于帮助团队从旧版“规格驱动开发文档集”迁移到新版“AI 前端交付工程规范”。

新版的核心变化不是换了一组名字，而是把重心从“文档前置”调整为：

- 统一输入
- AI 执行器接入
- 当前规则 / 当前规格表达
- 回写与资产复用

## 旧版与新版思路差异

| 旧版偏重点 | 新版偏重点 |
| --- | --- |
| 角色分工 | 确认机制 |
| 单线流程 | 标准模式 / 轻量模式 / 变更模式 |
| 文档前置 | 事实表达先行 |
| 工件名词 | 工程协议 |
| 沉淀归档 | 资产消费 |

# 快速上手

## 15 分钟先记住什么

这套体系的核心不是多写文档，而是建设AI 前端交付工程规范。

你只需要先记住三件事：

1. 页面交付走统一链路
2. 每个阶段有明确工件和 owner
3. 每次交付都必须沉淀资产

## 统一链路

`需求输入 -> Feature Brief -> Design Contract -> Page Spec -> UI -> Frontend -> Review -> Implementation Record -> Asset`

## 第一周最小动作

- 选一个真实页面
- 产出 `Feature Brief`
- 产出 `Design Contract`
- 产出 `Page Spec`
- 按 checklist 做一次 review
- 完成 `Implementation Record`
- 判断是否形成资产候选
